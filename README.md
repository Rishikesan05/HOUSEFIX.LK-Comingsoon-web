# HOUSEFIX.LK | Coming Soon Landing Page

This is the official **Coming Soon** landing page for **HOUSEFIX.LK**, a professional construction and smart infrastructure company based in Sri Lanka. The platform serves as a high-end pre-launch portal to build anticipation and capture early interest for premium building and automation services.

## 🌟 Key Features
- **Glassmorphic Aesthetic**: A modern, premium UI featuring animated "glow-blob" background elements and glassmorphic containers.
- **Micro-Animations**: Staggered fade-in animations for headers, badges, and service pills to enhance the user experience.
- **Service Highlights**: Interactive pills showcasing core business areas: Custom Construction, Renovation, Interior Design, and Smart Home Automation.
- **Notification Engine**: Integrated subscription form for users to request early-access notifications.
- **Responsive Architecture**: Fully optimised for mobile, tablet, and desktop devices using the modern 'Outfit' typography.

## 🛠️ Technical Stack
- **Core**: Semantic HTML5 and modern CSS3.
- **Animation**: CSS Custom Properties (Variables) and Keyframe animations for background dynamics.
- **Typography**: [Google Fonts](https://fonts.google.com/) (Outfit).
- **Icons**: Custom inline SVG architecture for performance and scalability.

## 📁 Repository Structure
- `index.html`: The main landing page structure and messaging.
- `style.css`: Comprehensive design system, animations, and responsive layout.
- `logo.png` & `logo-rounded.png`: Official branding assets.

## 🚀 Deployment
This project is a static website and can be deployed instantly to GitHub Pages, Vercel, Netlify, or any standard web server.
